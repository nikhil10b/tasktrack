## Task Management Application

### Completed Features:


1. **User Authentication** (Sign Up/Login using JWT authentication) 
2. **Task CRUD Operations** (Create, Read, Update, Delete tasks) 
3. **Task Status Management** (Pending, In Progress, Completed)
